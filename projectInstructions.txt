# 🎓 When Should You Learn useEffect?

After you can:

* Build small apps without copying
* Understand why re-renders happen
* Understand how state updates trigger UI changes

Then learn:

* useEffect
* dependency array
* lifecycle thinking

Because `useEffect` is about:
"Run this code when something changes."

If you don’t fully understand state yet,
`useEffect` will confuse you.
